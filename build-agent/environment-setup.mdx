## Prerequisites

Before building a voice agent, make sure the following tools are installed on your system:

- Python 3.9 or later
- pip (Python package manager)
- Git
- A code editor such as VS Code

You will also need access to a LiveKit server or LiveKit Cloud.

---

## Step 1: Install Python

Check if Python is installed by running:

```bash
python --version
```

or

```bash
python3 --version
```

If Python is not installed, download it from the official website.

<Card title="Download Python" icon="download" href="https://www.python.org/downloads/">
Install the latest version of Python.
</Card>

---

## Step 2: Create a Project Folder

Create a new folder for your voice agent project.

```bash
mkdir livekit-voice-agent
cd livekit-voice-agent
```

---

## Step 3: Create a Virtual Environment

Virtual environments isolate project dependencies.

Create one using:

```bash
python -m venv venv
```

Activate it.

Mac / Linux:

```bash
source venv/bin/activate
```

Windows:

```bash
venv\Scripts\activate
```

---

## Step 4: Install Dependencies

Install the required LiveKit agent framework.

```bash
pip install livekit-agents
```

You may also need additional packages for speech processing and AI models.

Example:

```bash
pip install openai
```

---

## Step 5: Configure Environment Variables

Voice agents often require API keys and configuration settings.

Create a `.env` file in your project directory.

```
LIVEKIT_API_KEY=your_api_key
LIVEKIT_API_SECRET=your_api_secret
LIVEKIT_URL=your_livekit_url
```

Environment variables help keep sensitive credentials secure.

---

## Step 6: Test Your Setup

Create a simple Python file.

```
agent.py
```

Add the following code:

```python
print("Voice agent environment ready!")
```

Run the script:

```bash
python agent.py
```

If everything is configured correctly, you should see:

```
Voice agent environment ready!
```
